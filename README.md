# FlopIQ

**Think Better. Stack Faster.**

Interactive poker decision training app that builds disciplined players through gamified drills, real math, and proven strategy.

## Features

- **Decision Drills** — 103 scenarios across preflop, flop, turn, and river
- **Smart Bet Typing** — Context-aware dropdown (C-Bet, Value Bet, Semi-Bluff, Bluff, etc.)
- **XP & Levels** — Progress from Fish to GTO Wizard across 8 ranks
- **Streak System** — Vegas-style firework celebrations at milestones with XP multipliers
- **Session Grades** — D (Dormant) through SSS (Unplayable) based on accuracy
- **Session Summary** — Category breakdown, personalized advice, streak tracking
- **Starting Hand Charts** — 3 reference charts (Loose, Tight, Facing Raise)

## Tech Stack

- React 19 + Vite 8
- No external UI libraries — pure inline styles
- LocalStorage for XP/level persistence
- Canvas-based particle animations

## Getting Started

```bash
npm install

```
